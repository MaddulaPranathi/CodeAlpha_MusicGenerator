# 🎶 AI Music Generator

An AI‑powered project that generates new music sequences using **MIDI datasets** and **deep learning models (LSTM/GANs)**.  
This app demonstrates preprocessing, training, and generation of music, producing `.mid` files that can be played or converted to audio.

---

## ✨ Features
- 📂 Preprocess MIDI files into note sequences.
- 🧠 Train deep learning models (LSTM/GRU/GAN).
- 🎼 Generate new music sequences.
- 💾 Save outputs as `.mid` files.
- 🔊 Play generated music using MIDI/audio libraries.
